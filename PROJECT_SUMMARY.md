# 🏛️ ملخص المشروع النهائي - نظام إدارة قاعات الأفراح

## 🎯 نظرة عامة

تم إنشاء نظام شامل ومتكامل لإدارة قاعات الأفراح يجمع بين قوة Laravel في الخلفية وحداثة Vue.js في الواجهة الأمامية، مع تصميم عربي أنيق ومتجاوب.

## 🚀 كيفية التشغيل

### تشغيل سريع
```bash
# على Linux/Mac
./start.sh

# على Windows
start.bat
```

### تشغيل يدوي
```bash
# تثبيت التبعيات
composer install
npm install

# إعداد البيئة
cp .env.example .env
php artisan key:generate

# إعداد قاعدة البيانات
php artisan migrate

# تشغيل الخادم
php artisan serve    # في نافذة منفصلة
npm run dev         # في نافذة أخرى
```
